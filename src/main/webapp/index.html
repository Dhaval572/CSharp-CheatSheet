C# Language Overview

What is C#?

C# (pronounced "C-sharp") is a modern, object-oriented programming language developed by Microsoft. It's part of the .NET ecosystem and is widely used for building various types of applications.

Key Features:
  • Type-safe: Compile-time type checking
  • Memory-managed: Automatic garbage collection
  • Cross-platform: Runs on Windows, macOS, and Linux
  • Rich ecosystem: Extensive .NET libraries
Sample C# Code
using System;

namespace HelloWorld
{
    class Program
    {
        static void Main(string[] args)
        {
            string name = "World";
            Console.WriteLine($"Hello, {name}!");
            
            // Basic operations
            int x = 10;
            int y = 20;
            int sum = x + y;
            
            Console.WriteLine($"Sum: {sum}");
        }
    }
}
